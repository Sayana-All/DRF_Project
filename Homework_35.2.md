# DRF_Project
### Создание платформы по онлайн-обучению с помощью Django REST framework

## Описание
В мире развивается тренд на онлайн-обучение. Но для веб-разработчика важно не только обучиться, но и знать, как реализовать платформу для онлайн-обучения. 
Поэтому новая задача касается разработки LMS-системы, в которой каждый желающий может размещать свои полезные материалы или курсы.

## Инcтрукции
Вы настроите удаленный сервер для развертывания и работы с веб-приложением, которое разрабатывали в домашках курса DRF. 
Создадите и настроите workflow-файл для GitHub Actions, чтобы автоматизировать процессы сборки, тестирования и доставки кода.

## Задачи
1. Настройте удаленный сервер для работы с веб-приложением, которое вы разрабатывали в рамках домашних работ на курсе DRF.

2. Создайте и настройте файл _GitHub Actions_ workflow, который будет:
    - Запускать тесты проекта автоматически при каждом push в репозиторий;
    - Автоматически деплоить проект на удаленный сервер после успешного прохождения тестов.

* **Дополнительное задание**
Разверните проект на удаленном сервере с использованием _Docker_:
    - Напишите **Dockerfile** для вашего проекта;
    - Настройте _GitHub Actions_ для автоматической сборки Docker-образа и его деплоя на удаленный сервер.
_Дополнительное задание, помеченное звездочкой, желательно, но не обязательно выполнять._
